
# She Can Foundation Website

A simple, clean landing page for a nonprofit organization that empowers women.

## 📂 Files in this project
* `index.html` - Holds the text, links, and structure.
* `style.css` - Holds the colors, fonts, and layouts.

## 🛠️ How to run it
1. Download both `index.html` and `style.css`.
2. Put both files into the **same folder** along with both images on your computer
4. Double-click `index.html` to open it and view the website in your web browser.

## 🎨 Colors used
* **Pink (#ff6b9d):** Main brand color for text, headers, and buttons.
* **Dark Gray (#2d3436):** Used for easy-to-read text and the footer.
* **Light Pink Tint (#fff5f9):** Soft background color.
